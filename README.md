Este programa...
se trata de hacer un Traductor. 
El traductor solo incluirá de 30 a 40 frutas. 
Usando TKinter crear una interfaz gráfica. 
En dicha interfaz el usuario deberá escribir la palabra de la fruta en español. 
Entonces, la aplicación debe mostrar el nombre de la fruta en inglés.
Para integrar la lista de palabras en español e inglés usar una estructura de Diccionario.
